# How to Use This Textbook

## Opening

You have forty-five free minutes between clinic and journal club. This chapter maps efficient paths through the curriculum so you spend time on the literacy layer that actually changes how you read a stroke-imaging model paper.

Read Parts I–III for identification of learning problems; Parts IV–V for predictors and representations; Parts VI–VII for sequential decisions, efficiency, graphs, and data operations. Use Navigation Pane headings. Symbols are defined on first use.

Learning Objectives: post-chapter competencies.
